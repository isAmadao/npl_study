## 🚀 快速开始

### 5分钟上手

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置API密钥 (编辑 config/system_config.json)

# 3. 运行演示
python main_demo.py
```

详细步骤请查看 [项目介绍](./introduction.md)

### 启动完整服务

```bash
# 终端1: 启动后端
uvicorn api.server:app --reload --host 0.0.0.0 --port 8000

# 终端2: 启动前端
cd frontend && npm install && npm run dev
```

访问:
- 前端界面: http://localhost:5173
- API文档: http://localhost:8000/docs

## 🎓 学习路径

### 新手入门

1. 📖 阅读 [项目介绍](./introduction.md) 了解项目背景
2. 🏃 运行 `main_demo.py` 体验游戏
3. 🎮 启动前端界面观战
4. 📝 查看 `logs/` 目录下的游戏记录

### 开发者入门

1. 🏗️ 研读 [架构文档](./architecture.md) 理解系统设计
2. 🔍 查看核心代码:
   - `engine/game_engine.py` - 游戏引擎
   - `agent/player_agent.py` - AI智能体
   - `roles/*.py` - 角色实现
3. 🧪 运行测试: `pytest tests/`
4. 🔧 尝试修改代码或添加新角色

### 进阶学习

1. 📈 阅读 [优化建议](./optimization.md) 了解改进方向
2. 🛠️ 参与项目开发，实现优化建议
3. 📊 分析游戏日志，研究AI决策模式
4. 🏆 对比不同LLM的表现
