# SemanticMessageHistory 文档

## 概述

语义消息历史管理模块，为 Agent 提供基于 Embedding 的对话上下文存储和检索能力。

## 核心功能

### 1. 会话管理
- 按 `session_id` 隔离不同对话
- 自动创建和索引会话
- 支持列出所有活跃会话

### 2. 消息存储
- 支持 user / assistant / system / tool 四种角色
- 自动生成消息 ID 和时间戳
- 可选的元数据附加

### 3. 语义检索
- 通过 Embedding 相似度从历史消息中检索相关上下文
- 支持自定义相似度阈值
- 适用于 RAG（检索增强生成）场景

### 4. 上下文窗口管理
- 基于 Token 数的上下文裁剪
- 内置中英文混合 Token 估算器
- 支持自定义 max_tokens

## 使用场景

1. **多轮对话**：保持对话的连续性
2. **RAG 检索**：从历史中找到与当前问题最相关的上下文
3. **Agent 记忆**：Agent 的长短期记忆管理
4. **会话分析**：统计会话的角色分布和 Token 消耗
