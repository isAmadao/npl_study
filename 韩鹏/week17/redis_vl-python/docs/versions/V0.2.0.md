# V0.2.0 — EmbeddingsCache 模块

## 版本目标

实现 EmbeddingsCache 模块：对文本的嵌入向量做精确 hash 缓存，避免重复计算相同的向量。完成前端展示卡片，显示缓存命中/未命中状态及向量摘要。

## 前置条件

- V0.1.0 已完成（项目脚手架可运行）
- Redis Stack 容器可用
- HuggingFace 模型可在本地缓存中找到

### 任务 2：实现 EmbeddingsCache 模块

- [x] 创建 `app/modules/embeddings_cache.py`
  - 类 `EmbeddingsCache`
  - `__init__(self, redis_client, name="embeddings", ttl=None)`
  - `set(texts, embeddings)` — 以 hash(text + model) 为 key 存储向量
  - `get(texts)` — 批量获取缓存向量，返回 dict {text: vector or None}
  - `exists(text)` — 检查单个文本是否存在缓存
  - 向量序列化：float 列表用逗号拼接存为 Redis String
  - key 格式：`embedcache:{name}:{md5_hash}`
  - 支持 TTL 设置
