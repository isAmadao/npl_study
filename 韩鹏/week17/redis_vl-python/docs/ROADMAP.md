## 开发触发机制

用户发送 **“开发下一个版本”** 时：

1. 读取本文件，找到状态为 `⏳ 待开发` 的版本。
2. 打开对应版本文档 `docs/versions/VX.X.X.md`。
3. 按版本文档中的任务清单逐项实现。
4. 开发完成后运行测试并修复回归。
5. 更新本文件中当前版本和测试结果。
6. 如果所有版本都已完成，则明确告知用户当前已到最终版本。
