# AI狼人杀 — 开发目标与要求

## 项目概述

基于多Agent协作框架，构建一个能够自主完成信息不对称博弈的AI狼人杀AgentTeam系统。核心在于多智能体的协作/对抗与交互机制设计。

## 核心业务要求

1. **多角色Agent**：每个Agent根据其扮演角色（狼人、预言家、女巫等）拥有独立的目标、策略与行动空间
2. **信息隔离**：严格信息隔离约束下进行推理、发言与决策
3. **对局引擎**：完整的对局引擎，驱动回合流转与胜负裁决
4. **结构化日志**：输出结构化日志实现全程可观测
5. **前端观战**：Vue3观战UI，支持AI对战
6. **自演化**：Agent从「通用Agent」演化为「狼人杀多角色Agent」

## MVP范围约束

- 角色数量：6人标准局（2狼人 + 1预言家 + 1女巫 + 2村民）
- LLM推理：使用 DeepSeek v4-flash API
- 前端：单页面简洁UI，观战 + 启动新游戏
- 不涉及：猎人、守卫、白痴等扩展角色；人机混战；高级自演化

## 技术约束

| 项目 | 要求 |
